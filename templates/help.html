Molytica Chemical Analyser

Yardım / Help

Türkçe: Bu, Chemical Data Analyzer kullanımı için kapsamlı bir rehberdir. Bu araç, kimyasal veri analizi yapmanızı, tahminler oluşturmanızı ve görselleştirmelerle raporlar hazırlamanızı sağlar.

1. Sistem Gereksinimleri / System Requirements

Türkçe: Aşağıdaki sistem gereksinimleri, yazılımı sorunsuz bir şekilde çalıştırabilmeniz için gereklidir:

  • Tarayıcı / Browser: Google Chrome, Mozilla Firefox, Safari, veya Microsoft Edge'in en son sürümleri.
  • İşletim Sistemi / Operating System: Windows, macOS veya Linux.
  • Ek Araçlar / Additional Tools: Python (Veri analizi için) ve Node.js (Arka uç JavaScript işlevselliği için) gereklidir.
  • Veri Yükleme / File Upload: CSV veya Excel dosyaları ile veri yükleme özelliği desteklenir.

2. Desteklenen Dosya Formatları / Supported File Formats

  • CSV: Tabular veriler için standart format. Veri analizi yapabilmek için her kolon başlığı düzgün bir şekilde etiketlenmelidir.
  • Excel: Excel dosyalarını CSV formatına dönüştürerek yükleyebilirsiniz.
  • Manuel Veri Girişi / Manual Data Entry: Veri girişi yapmak için el ile veri girme özelliği mevcuttur.

3. Nasıl Kullanılır? / How to Use?

Türkçe: Yazılımın kullanımı adım adım aşağıda açıklanmıştır:

3.1. Veri Yükleme / Data Upload

  • Manuel Veri Girişi / Manual Data Entry: "Manuel Veri Girişi" butonuna tıklayın ve verilerinizi elle girin. Bu seçenek, küçük veri setleri için idealdir.
  • CSV Dosyası Yükleme / Upload CSV File: "CSV Dosyası Yükle" butonuna tıklayarak bir CSV dosyasını yükleyin. Verinizin doğru formatta olduğundan emin olun.
  • Excel Dosyası Yükleme / Upload Excel File: "Excel Dosyası Yükle" butonuna tıklayarak bir Excel dosyasını seçin. Sistem, Excel dosyasını otomatik olarak CSV'ye dönüştürerek yükleyecektir.

3.2. Veri Analizi / Data Analysis

  • Tahmin Yap / Make Prediction: Yüklediğiniz verilerle tahmin yapmak için "Tahmin Yap" butonuna tıklayın. Bu işlem, verileri analiz eder ve kullanıcıya tahminler sunar.
  • Görselleştirme / Visualization: Verilerinizi daha iyi anlamak için "Görselleştirme" butonuna tıklayın. Çeşitli grafikler ve diyagramlarla verilerinizi görsel hale getirebilirsiniz.
  • Rapor Oluştur / Generate Report: "Rapor Oluştur" butonuna tıklayarak analizinizin raporunu otomatik olarak oluşturabilirsiniz. Rapor, verilerinizin analizini ve tahmin sonuçlarını içerecektir.

4. Veri Seti Gereksinimleri / Dataset Requirements

  • Zaman Bilgisi / Time Information: Eğer zaman serisi tahmini yapıyorsanız, verinizin tarih/saat bilgilerini içermesi gerekmektedir.
  • Kategorik Özellikler / Categorical Features: Verinizde kategorik değişkenler (örneğin: kimyasal tür, bölge) olmalıdır.
  • Sayıysal Özellikler / Numeric Features: Verinizde sayısal değişkenler (örneğin: sıcaklık, yoğunluk) bulunmalıdır.
  • Eksik Veriler / Missing Data: Verinizde eksik veriler bulunmamalıdır. Eksik verileri tamamlamalı veya çıkarma işlemi yapmalısınız.
  • Kolon Başlıkları / Column Headers: Her bir kolonun başlığı açık bir şekilde etiketlenmelidir.

English: This is a comprehensive guide to using Chemical Data Analyzer. This tool allows you to perform chemical data analysis, generate predictions, and create reports with visualizations.

1. System Requirements

The following system requirements are necessary to run the software smoothly:

  • Browser: Latest versions of Google Chrome, Mozilla Firefox, Safari, or Microsoft Edge.
  • Operating System: Windows, macOS, or Linux.
  • Additional Tools: Python (for data analysis) and Node.js (for backend JavaScript functionality) are required.
  • File Upload: Support for uploading data via CSV or Excel files.

2. Supported File Formats

  • CSV: The standard format for tabular data. Each column header must be properly labeled for analysis.
  • Excel: You can upload Excel files by converting them to CSV format.
  • Manual Data Entry: You can manually enter data through the provided option.

3. How to Use?

  • Manual Data Entry: Click the "Manual Data Entry" button and enter your data manually.
  • Upload CSV File: Click the "Upload CSV File" button to upload a CSV file.
  • Upload Excel File: Click the "Upload Excel File" button to select an Excel file.

4. Dataset Requirements

  • Time Information: If you are doing time series forecasting, your dataset should include date/time information.
  • Categorical Features: Your dataset should include categorical variables (e.g., chemical type, region).
  • Numeric Features: Your dataset should include numeric variables (e.g., temperature, concentration).
  • Missing Data: Your dataset should not contain missing data. You should fill or remove missing values.
  • Column Headers: Each column should be clearly labeled with a header.
